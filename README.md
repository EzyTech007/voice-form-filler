# Voice Form Filler

A web application where you can upload any kind of form and it will ask you questions to fill those fields using voice interaction.

## Built with React + Vite

This project uses React with Vite for fast development and HMR (Hot Module Replacement).

### Available Plugins

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)
